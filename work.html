Using Teachable Machine to to identify audio input, this interactive web app allows a user to input a note they wish to play, then uses audio input to identify if the note the user plays is correct. If not correct, the note they are currently playing is identified and the instructions for playing the correct one are displayed.

Interactive Flute Educational Web App

A web app that combines the the look and convenience of digital art with the intuitive and classic feel of arcade game controls, this web app and physical control board allow the user to draw on-screen with a variety of colours, brush sizes, and opacities. Constructed of a mix of conductive materials, non-conductive materials, and wire, the control board is connected to a custom-mapped Makey Makey in order to draw in P5.js.

Digital Painting Web App and Control Board

Using a series of custom graphs paired with simple paragraphs, this graphic illustrates the relationship between the success of software company Zoom and the sudden presence of the Covid-19 virus. Using databanks to collect data, Microsoft Excel to analyze it and create graphs, and Adobe Illustrator for stylizing, this graphic tells the narrative that Zoom's staggering success can be attributed to being at the right place at the right time.

Right Place, Right Time: Covid-19 and the Success of Zoom
